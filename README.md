# Recipe Vault

## Description

Recipe Vault is a web application for storing, organizing, and browsing recipes.

## Current Functionality

- Reusable RecipeCard component
- Dynamic recipe data passed through component props
- Search recipes by title, cuisine, and description
- Display a no-results message when no recipes match the search

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
